3. ⚙️ Configuration des scènes (Javascript)

C'est la partie logique du code. Le script définit un dictionnaire de scènes, où chaque scène correspond à une pièce de la visite.

Structure logique du script de configuration

L'objet principal `pannellum.viewer()` initialise la visionneuse. Il prend deux arguments : l'ID du conteneur HTML ("panorama") et un objet de configuration.

  • `default` : Contient les paramètres globaux (scène de départ, chargement automatique).
  • `scenes` : Contient la définition de chaque pièce (`hall`, `cafe`, etc.).

Objectif : Définir la scène par défaut (`hall`) et la structure de la scène. 

const viewer = pannellum.viewer("panorama", {
  default: {
    firstScene: "hall", 
    autoLoad: true,     
  },
  scenes: {
    hall: {
      title: "hall",
      type: "equirectangular",    // Format standard 360
      panorama: "./img/hall.png", // Chemin vers l'image
      autoRotate: -2,             // Rotation automatique lente
      hotSpots: [ /* ... points de navigation ... */ ]
    },
    // ... autres scènes ...
  }
});
